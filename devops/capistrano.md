# Capistrano简易教程

- 设置Capistrano

```
// 安装capistrano
gem install capistrano

// 安装时可能会出现错误，执行以下步骤
//（ERROR: ...Unable to download data from http://ruby.taobao.org/ - bad response ...）
gem sources --remove http://ruby.taobao.org/
gem sources -a https://ruby.taobao.org/
gem sources -l // 查看源对不对
// 重新安装
  

```
